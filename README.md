# Hangman Game (Python) 🎮

A classic console-based **Hangman** game developed to demonstrate core Python concepts: logic loops, string manipulation, and ASCII art integration.

## 🚀 Features
- **Randomized Logic**: Secret words are chosen randomly from a built-in library.
- **ASCII Visuals**: Real-time visualization of the hangman's progress.
- **Input Validation**: Handles duplicates and non-alphabetical characters.
- **Clean Code**: Optimized and commented for educational purposes.

## 🛠️ Tech Stack
- **Language**: Python 3.x
- **Module**: `random`

## 📖 How to Run
1. Ensure you have Python installed.
2. Download `main.py`.
3. Run the following command in your terminal:
   ```bash
   python main.py
